Numerous challenges presented themselves throughout the development process. Some examples:

  • Proper form va_on_submit() method was invalid. Documentation led me to an alternate method for validating form fields.
  • Custom CSS:CDN meant that I had no easy way to manipulate built-in values. A good deal of searching led me to the code I needed to calculate color hex values and create my own btn-custom class.
  • SMTP: my w Several searches on stackoverflow eventually revealed that AWS throttles outbound EC2 traffic on port 25. Explicitly defining a different port for SMTP traffic has restored email functionality.

Habit Tracker

An API-based Python GUI app

  • Python
  • guizero, tkinter, requests

After working with the Pixe.la APIs through the console during a boospent coding and learning on a pixel graph. I reused the API-related code that I had authored during the bootcamp project and built the GUI from scratch.

This was my first codheavily on the guizero documentation for reference.

Some notable challenges:

  • Image resizing: I wantedarching led me to discover that guizero required the Pillow library to scale images. Installing the Pillow package created the desired outcome.
  • Error handling: PI. I learned how to use a try/except block to catch invalid entries, and used the datetime module and strftime to convert the date entry into an appropriate date object.
  • GUI operation: I would use tkinter methods with guizero to solve this challenge. In the case of resetting radio buttons, this involved creating a list of the options and calling tk.select() on the desired list index.
  • Including the graph: Pixe.la graphs are rendered in SVG format in a browser window. I attempted to fmage processing application. I did not like the idea of having to include a separate program with this app, so I settled for a button that opens a link to the selected graph in a new browser tab.

For the time being, this application is complete. Future improvements may include: creating an executable using PyInstallerif a library for doing so becomes available.
